# hardware Setup
in this project i'm using MG996R servo motor and Ultrasound distance HC-SR04

## Wire

| MG996R | ESP8266 |
|--------|---------|
| GND    | GND     |
| 5V     | 5V      |
| PWM    | D7      |

| HC-SR04  | ESP8266 |
|----------|---------|
| GND      | GND     |
| 5V       | 5V      |
| ECHO     | D1      |
| TRIG     | D2      |
